关闭SIP的方法

  1. 首先, 进入Mac的Recovery Mode.
  2. 进入终端: [实用工具]-[终端].
  3. 终端中输入: csrutil disable.
  4. 重启之后, 输入csrutil disable, 查看SIP是否被disable.
  5. 之后, 在终端运行: sudo nvram boot-args=-arm64e_preview_abi.
  6. 然后重启.

下载完yabai之后

在/etc/sudoers.d中, 创建yabai, 写入:

  • user: whoami的输出.
  • hash: shasum -a 256 $(which yabai)的输出.
  • yabai: which yabai的输出.

注意, yabai更新之后, hash的值会改变, 需要重新修改.
